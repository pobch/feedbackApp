## GetFeedback App
Stack :
- Front-end
  - React.js
  - Redux
  - Redux-thunk
  - React-router
  - Redux-form
  - Axios
  - MaterializeCSS
- Back-end :
  - Node.js
  - Express.js
  - Passport.js
  - MongoDB
- Services :
  - Stripe
  - SendGrid
______
